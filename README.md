## Git

### Branch

- main : 서비스 운영 브랜치
- dev : 개발중인 브랜치
- feat/기능명 : 기능 개발 브랜치

<br>

### Commit Message

| Message  |               설명                |
| :------: | :-------------------------------: |
|   feat   |         새로운 기능 추가          |
|   fix    |             버그 수정             |
|   docs   |             문서 수정             |
| refactor |           코드 리팩터링           |
|  style   |            디자인 변경            |
|  chore   | 빌드 부분 혹은 패키지 매니저 수정 |
|  remove  |          코드(파일) 삭제          |
|  rename  |       파일 혹은 폴더명 수정       |
| release  |               배포                |
